## Shape

- Small radius: 12px
- Base radius: 16px
- Large radius: 24px
- Pill radius: 999px

## Touch

- Minimum comfortable touch target: 40px
- Preferred mobile target: 44px
- Default input height: 44px
- Large control height: 52px

## Typography

- UI Latin/display: Quicksand
- Chinese system fallback: PingFang SC / Hiragino Sans GB / Microsoft YaHei
- Core package exposes font variables but does not bundle font files. The docs site may load font files for brand presentation.
